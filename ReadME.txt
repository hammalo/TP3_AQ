Pour le test le plus simple où le jeu est fermé, nous allons vérifier l'état du jeu pour confirmer qu'il est fermé. Ce test est un exemple de test d'état, car nous vérifions simplement l'état du jeu après son exécution sans vérifier ses interactions avec d'autres objets ou composants. Dans ce cas, nous nous attendons à ce que la méthode jouer du jeu lève une exception JeuFermeException lorsque le jeu est fermé. Nous n'interagissons pas directement avec d'autres objets tels que les dés ou la banque pour ce test.
